# Ecommerce

Full stack ecommerce website created using the mern stack.

#### FrontEnd

- React Redux ToolKit
- React Toastify
- Fully mobile responsive

#### BackEnd

- Express
- MongoDB
- Mongoose
- Axios
- Hashed passwords through bcrypy
- User authentication through JWT
